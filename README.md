# Power-BI---GA4-Ecommerce-Insights
Análisis de GA4 (Google Store) con Power BI. Limpieza de datos crudos mediante Power Query: conversión de fechas (YYYYMMDD) y timestamps (Unix) a formatos legibles. Modelado de eventos para insights de e-commerce y comportamiento del usuario. Incluye diccionario de datos y ETL avanzado.
